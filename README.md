# Zynovix AI 1.0

Official static website for Zynovix AI, ready for GitHub Pages.

## Structure
- `index.html` — Homepage
- `tools.html` — AI tools page
- `learn.html` — Learn AI page
- `blog.html` — Blog page
- `about.html` — About page
- `contact.html` — Contact page
- `assets/css/style.css` — Main styles
- `assets/js/script.js` — Menu and reveal animations
- `assets/logo/` — Logo and favicon
- `robots.txt`, `sitemap.xml` — SEO files

## Deploy
Upload all files and folders to the root of `Zynovix/zynovix.github.io`, then commit with:

`Launch Zynovix AI 1.0`
